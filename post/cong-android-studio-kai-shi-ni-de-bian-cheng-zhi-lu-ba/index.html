  2. 973字

@牛顿：我和那些孩童没有什么区别，我不过是一个在沙滩上拾起贝壳研究的孩子，在真理之海面前瑟瑟发抖。

建立你的第一个工程（project）

新建工程

  • 请注意：在 Android Studio 4.0 预览版中我无法启动离线模式，所以接下来的内容基于 3.5.3 版本。
    包名：用网址以区分应用程序，请把 example.android.com 改成你自己的网站。
    语言：请将 Kotlin 改为 Java。使用 Kotlin 开发 Android 应用，使用 Kotlin 更快地编写更出色的 Android 应用。Kotlin 是一种静态类型的现代编程语言，不仅可以帮助您提高工作效率，而且还能让开发者工作起来更开心。
    最小SDK等级：你想创建的应用程序最低支持的安卓版本，请注意，如果SDK太小你可能无法使用很多新功能。Android 版本历史 | 维基百科

这个工程文件涵盖了 Java、XML、图片、音频之类的各种文件。

解决来自 gradle 奇奇怪怪的问题（难点）

第一次使用时 gradle 需要同步一段时间，有点久。你可以看到下面这个通知栏，当它不再有一个圈转来转去的时候就完成了，这时 Build 栏里也没有圈圈了，于是我们可以把它关掉。左边是类似于资源管理器的树状文件结构，请找到并双击 app→res→layout→activity_mail.xml，把右边的更新内容也关掉，就可以看到 Preview 了。中间是类似于记事本的编辑区，请点击下方的 Text 查看和编辑代码。如果你遇到有关 gradle 的问题，这里有一些解决方法。来源：https://cloud.tencent.com/developer/article/1523424

开始吧！在你的手机或者模拟器上运行（难点）

复习一些知识

xmlns 用于声明以下内容适用于安卓，因为 XML 语言也运用于制作网页或者其他功能。http://stackoverflow.com/questions/1181888/what-does-xmlns-in-xml-mean
setting→Editor→Font 调整编辑器的字体和大小

  • 这里给你一点建议。为了流畅的使用 Android Studio，最好把其他程序都关掉，将电脑（尤其是笔记本）调整至高性能模式。然后在手机上查看此教程。
    然后让我们在模拟器或者手机上运行吧。
  • 小建议：如果你的电脑卡得出翔，那最好还是不要使用模拟器了，用手机吧。比如我用三星a60，因为我的电脑是差不多十年前的ASUS，尽管4G内存，处理器也力不从心。

在手机上运行

请参考：https://developer.android.google.cn/studio/run/device
确认绿色的运行左边的框里写的是你的手机型号，那表示连接成功了。
成功运行！
如果没有成功也不要灰心，试着在网上找找答案。有一种说法是要在设置的Android SDK 中安装手机对应的版本，但是我的安卓9也没安装啊😂，反而是装了安卓10。还有一种说法是要到设备管理器中找到 ADB 设备并右键更新。

  • 我的运行方式：安装三星的驱动，打开USB调试，连上电脑一会儿就好了😊。
    尝试完成后请别忘记中止。
