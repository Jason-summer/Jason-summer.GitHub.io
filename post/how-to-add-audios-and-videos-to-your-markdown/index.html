如何在 Markdown 中插入视频和音频

2020-04-04 2 min read # Markdown

本篇文章的受众是那些没有 xml 基础，并且像我一样🤣只会基础 Markdown 的小白。

插入音乐

方法一、使用 html

<audio id="audio" controls="" preload="none">
      <source id="m4a" src="https://m701.music.126.net/20200404143533/acbed1605c603887dd9a181ffd6d57d4/jdyyaac/005a/545e/0009/70f3057ebac49b803e727c14ce6e5a83.m4a">
      </audio>

<audio>标签常用属性如下表

属性 值 描述
autoplay autoplay 添加该属性后，音频会自动播放
controls controls 设置后，显示控件，如播放按钮、音量
loop loop 添加该属性后，当音频播放结束后会重新开始播放
preload preload 音频显示页面加载，准备播放，如已添加autoplay,则忽略该属性
src url 播放音频的URL（地址）。

方法二、使用各个音乐平台提供的插件

网易云音乐提供了将音乐生成外链（某些版权歌曲不可）的插件。访问网易云的网页，在某首歌曲的详情页有生成外链播放器，接下来只要把获得的代码插入 Markdown 中就行了。

插入视频

<video id="video" controls="" preload="none" poster="http://img.blog.fandong.me/2017-08-26-Markdown-Advance-Video.jpg">
      <source id="mp4" src="http://img.blog.fandong.me/2017-08-26-Markdown-Advance-Video.mp4" type="video/mp4">
      </video>
      ```
