“避风港”的诞生日记

“避风港”是只属于我一个人狂欢的 party宁静的、安全的港湾，在我烦闷的时候（将会）给我带来快乐。现在，如果你看到了这里，那请姑且让我占用你一点时间，来讲讲这个“避风港”的建立过程吧🙂。

初识博客

2020/4/3，从大概小学五六年级的时候开始，我就使用QQ空间作为我的博客😂，我在QQ空间的日志那里写小说，后来还被我的同学看见了（致命打击）。真正开始使用QQ空间是在刚上初中的时候，那时候我发了两句歌词来怀念我小学毕业（十分做作的行为），后来初中也没少发说说。上了高中更是频繁，特别是还有一堆社团活动转发什么的……

知识焦虑

挺久之前看少数派时有见过讲如何建立自己的个人网站和博客的教程，但是当时我没有采纳。事实是，当时我对少数派上说的很多东西都感到挺疑惑的，比如什么“工作流”（这个倒是在高中电视台里学习 Pr 的时候认识了）啊，“知识库”啊，“数字焦虑”和“数字健康”啊……然后当时我的想法是，这都什么玩意，瞎 JB 扯淡吧。现在我要为当时的无知道歉🙇。

当我（周末和在疫情期间）被QQ上疯狂闪烁的小红点和99+差点搞成傻逼，我开始理解数字焦虑；当我刷完酷安开始刷少数派，然后是品玩、36kr和爱范儿……看一下时间发现到点睡觉了，再看看我电脑和手机前面的一沓沓没写完的作业📕，我是关注时间管理；当我发现我的账户体系一片混乱（尽管这也和目前国内的账户生态有关），有时候打开一个网站我甚至不知道我应该用手机号还是QQ登录（因为我不知道当时我是用什么注册的，甚至可能有些网站一开始没有QQ登录的选项后来又加入了，或者把入口埋得很深），而且即便打开了我也不知道我想要找的文件在哪一个网站的哪一个文件夹里面（更或者，每次在一台新设备上登陆 GitHub 或者 Steam 这种密码要求较高的网站的时候我总是想不起密码是什么😋，然后依稀记得我把密码记在了某个地方，但是我却不记得记在了哪里），我开始渴望知识库的构建。

当我亲身经历，我才发现这些东西不是空谈。我在学校上学的时候，每逢周六放假我总是不愿意回家，因为我知道一旦我回去，迎接我的会是海量的信息，我不知道从哪里开始看起。我甚至享受脱离手机和网络的生活，享受纸质书和图书馆的实体书带给我的快乐。

避风港湾

为了在互联网世界给自己留一块心灵空间，我先是尝试在语雀上建立我的知识库，同时印象笔记也是我的知识库的一部分，毕竟剪藏太好用了，就是每个月 60m 有点难以忍受。一天后（没错也就是今天😉），我遵循之前在少数派看到的文章开始尝试建立博客，然后就发现了 Gridea 这款宝藏般的博客管理软件，还有教程，跟着一步步走，在 Github Pages 上建立了网站，然后成功搭载了 Gridea 的服务。又用了一两个小时研究 Gitalk 评论系统（目前应该还是在被限流当中，你可以查看目前顶置的告示了解详情），终于上线了这个博客“夏元素的避风港”。

目前还不是很完善，只能作为 Github Pages 的一个子域名使用，所以接下来，我需要找到一个免费🤭的网址链接上我的博客。想想虽然 Github 有被墙抽风的可能，但我着实不太想被莫名其妙的审查然后“网站爆炸”，所以还是不用腾讯收购的 Coding（其实我挺喜欢的）或者 Gitee 了。

故事尾声

讲个人经历总得有个结尾吧😏？那么，希望在接下来的时光里我能继续我探索世界的旅程。如果在你看到了我的这篇文章后颇有感概（或者哪怕只有那么一点），那也不枉我写的这篇文章了。作为第一批使用的用户，我想起给未来写封信这个 application 的开屏图上的话：

多年以后，愿你不负所期。
